## Description

使用 BILIBLI 的API.
目前只能提供简单的番剧查看.
还有每周更新番剧也能看哦!

## Usage

`Ctrl+Shift+P` 调出控制栏后,输入`Bangumi Open`或 `Bangumi Open: Week Bangumi` 即可打开视图.

`Next Page` 和 `Back Page` 提供翻页功能.

`Next Page` 可用 `Ctrl+p` 代替

`Back Page` 可用 `Ctrl+o` 代替
